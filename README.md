# weather-app
## アプリ概要
- 現在の天気を確認できるwebアプリケーションです。

## 使用技術
|名称|バージョン|
|---|---------|
|react|18.2.0|
|typescript|5.0.2|
|vite|4.3.9|
|sass|1.63.3|
|axios|1.4.0|
|lottie-react|2.4.0|
|eslint|8.38.0|
|prettier|2.8.8|

## 天気API
- open weather map のAPIを利用しています。
- 環境変数として、VITE_WEATHER_API_BASE_URLとVITE_WEATHER_API_KEYを指定します。
